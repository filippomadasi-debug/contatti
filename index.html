Sincronizzazione cloud (Supabase)

Per avere gli stessi dati su telefono e PC: 1) crea un progetto gratuito su supabase.com (regione EU) · 2) nel SQL Editor esegui lo script qui sotto · 3) da Project Settings → API copia URL e chiave anon public e incollali qui. Ripeti il punto 3 su ogni dispositivo.

Script SQL da eseguire in Supabase 
create table if not exists contacts (
  id uuid primary key,
  cliente text, nome text, cognome text, ruolo text, note text,
  created_at timestamptz default now());
create table if not exists events (
  id uuid primary key,
  data date, tipo text, tono text, argomenti text, note text,
  contact_ids jsonb default '[]',
  created_at timestamptz default now());
alter table contacts enable row level security;
alter table events enable row level security;
create policy "anon all c" on contacts for all using (true) with check (true);
create policy "anon all e" on events for all using (true) with check (true);

⚠️ Chiunque abbia la chiave anon può leggere i dati: non condividerla.
